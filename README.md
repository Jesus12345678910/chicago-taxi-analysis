# Chicago Taxi Analysis

## Project Overview
SQL and Python analysis of Chicago taxi trip patterns and weather impact on ride duration using relational database queries, data modeling techniques, and data visualization.

## Objectives
- Analyze taxi trip patterns in Chicago using SQL queries
- Evaluate weather conditions impact on trip duration
- Identify peak usage patterns and route optimization opportunities
- Create data visualizations for key insights

## Technologies Used
- **SQL**: Queries with JOINs, CASE statements and WHERE conditions
- **Python**: Data analysis and visualization
- **Jupyter Notebook**: Interactive analysis and reporting
