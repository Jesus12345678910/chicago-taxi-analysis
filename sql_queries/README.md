## Query Files Description
- `01_data_exploration.sql`: Initial data exploration queries
- `02_company_analysis.sql`: Company performance analysis

- `04_advanced_queries.sql`: Complex analytical queries

## Tables Overview
- **cabs**: Taxi company information
- **trips**: Individual trip records
- **neighborhoods**: Geographic areas
- **weather**: Weather conditions data
